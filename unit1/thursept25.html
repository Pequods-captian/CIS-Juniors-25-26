TML lesson 3: text formatting and ordered lists and unordered lists

Basic text formatting

Use the strong tag or the B tag to make something bold

The diffrence between the B tag and the strong tag

The strong tag is an HTML semantic tag, There's a meaning using the strong tag, It tells the browser that There's something important about that text insid of the strong tag

The B tag means nothing, It simply bolds text.

The em tag and the i tag

The em tag and i tag, make the text inside of them italic, again the diffrence is that the em tag is a semantic tag and tells the browser something is special about that text and the i tag just turns it italic.

The U tag

It just underlines text.

Html Lists

There are three types of html lists, The ordered list, The unordered list, and The description list otherwise known as the Dictonary list

The Ordered List

  1. Uses the OL tag
  2. It is numbered
  3. Uses Li tag to add stuff to the list

The Unordered List

  • Uses UL for unordered list
  • Uses bulletpoints instead of numbers
  • Also uses Li tag to put stuff in it
